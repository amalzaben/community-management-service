# 🌐 Community Management System (Spring Boot)

A backend application built with **Spring Boot** to explore and implement core backend engineering concepts including **Spring Data JPA**, **Spring Security**, and **microservices architecture**.

This project focuses on building a secure, scalable RESTful API with layered architecture and inter-service communication.

---

## 🚀 Objectives

This project was developed to practice and understand:

- 🔐 Authentication & Authorization using **Spring Security**
- 🧩 Building multiple **filter layers** for request processing
- 🗄️ Database interaction using **Spring Data JPA**
- 🏗️ Designing a **microservices-based architecture**
- 🔗 Inter-service communication using **WebClient**
- 👥 Integrating with an external **User Management Service**

---

## ⚙️ Features

- 🔐 Secure login system with role-based access control
- 🧾 RESTful APIs for managing community-related data
- 🧱 Layered architecture (Controller → Service → Repository)
- 🛡️ Custom security filters
- 🔄 Communication with external microservice (User Service)
- 📦 Clean and modular project structure

---

## 🛠 Tech Stack

- **Backend:** Spring Boot  
- **Security:** Spring Security  
- **Database:** JPA 
- **Architecture:** Microservices  
- **HTTP Client:** WebClient
